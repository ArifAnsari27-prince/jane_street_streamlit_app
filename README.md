# Jane Street India — Streamlit Research Dashboard

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Included
- Streamlit app with polished dark-theme forensic dashboard
- Mechanistic simulator based on the uploaded quantitative note
- Footprint scoring engine for research attribution
- Sections on market microstructure and frontier extensions to South Korea block-trade regimes and crypto venues
- CSV data room with download buttons

## Data
The app reads local CSV files from `data/`.
